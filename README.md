# AWS CF TEMPLATE EC2

Foobar is a Python library for dealing with word pluralization.

## Installation

Install AWS CLI

## Usage

```bash
ws cloudformation deploy --template-file ec2-httpd-template.yaml --stack-name test
aws cloudformation delete-stack --stack-name test

```
